
# Projeto de Análise de Dados - Alura Store

## Descrição

Este projeto realiza a análise de dados de vendas de quatro lojas fictícias da Alura Store, com o objetivo de entender o desempenho de cada loja. A análise inclui métricas como faturamento, categorias de produtos, avaliações dos clientes, produtos mais vendidos, e custo médio de frete. A partir dessas análises, geramos visualizações que ajudam a identificar padrões e tomar decisões sobre qual loja o Senhor João deve vender.

## Tecnologias Utilizadas

- **Python 3.x**
- **Pandas**: Para manipulação e análise de dados.
- **Matplotlib**: Para visualizações gráficas.
- **GitHub**: Para versionamento e compartilhamento do código.
